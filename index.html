Tο Express είναι ενα framework και παίζει σημαντικό ρόλο στη πλατφορμα Node.js,υποστηρίζοντας την ασύγχρονη διάδραση με τις παραδοσιακές και NoSQL μεθόδους βάσεων δεδομένων.

Το Node.js είναι μια πλατφόρμα ανάπτυξης λογισμικού (κυρίως διακομιστών) χτισμένη σε περιβάλλον Javascript. Στόχος του Node είναι να παρέχει ένα εύκολο τρόπο δημιουργίας κλιμακωτών διαδικτυακών εφαρμογών.

Η JavaScript είναι διερμηνευμένη γλώσσα προγραμματισμού. Είναι μια γλώσσα σεναρίων που βασίζεται στα πρωτότυπα ,είναι δυναμική, με ασθενείς τύπους και έχει συναρτήσεις ως αντικείμενα πρώτης τάξης.

Η jQuery είναι μια βιβλιοθήκη JavaScript σχεδιασμένη να απλοποιήσει την υλοποίηση σεναρίων στην πλευρά του πελάτη της HTML και υποστηρίζει πολλαπλούς φυλλομετρητές Ιστού.

Το MongoDB είναι ένα πρόγραμμα βάσης δεδομένων. Κατατάχθηκε ως πρόγραμμα βάσης δεδομένων NoSQL και ειναι κυριως μη σχεσιακη, το MongoDB χρησιμοποιεί έγγραφα τύπου JSON.
